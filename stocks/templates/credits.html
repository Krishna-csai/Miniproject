This is a website created by Krishna Tyagi.

This website can show the real time prices and some other important information related to stock market Companies of India.

This website is based on a Python framework named Django. Django is full-stack web development framework based on python and uses python as its backend language.

This website shows data in real time by taking its data from the Official website of NSE India.

When the user asks for data on this website, the website calls for a python function in the backend which then obtains data from the official website of NSE and then returns an Json file as a response. Then the website fetches that response and converts those Json objects into useful data and then displays it to the user.

The python library used on this website to get data from NSE website is NSETOOLS.
